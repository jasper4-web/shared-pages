React Bits

Animated UIFree · MIT + Commons Clause

The gold standard for copy-paste animated React components.

  • 130+ free animated React components: text effects like BlurText and scramble reveals, animated backgrounds, and interactive UI flourishes.
  • Every component ships in four variants — JavaScript or TypeScript, plain CSS or Tailwind — so it drops into almost any React stack.
  • Installs via the shadcn CLI or plain copy-paste, which makes it trivially consumable by Claude Code.
  • Paste a component, wire the props, and you have premium motion design without writing GSAP or Framer code by hand.
  • Also bundles three free companion generators: Background Studio, Shape Magic, and Texture Lab.
  • 43.2k GitHub stars and 1,421 commits — one of the fastest-growing UI projects of 2025 and still actively maintained.
  • Free for personal, client, and commercial work; you just can't resell the library itself.
Why it wonIt's the benchmark this whole directory is calibrated against — and it holds up under inspection, growing from 110+ to 130+ components since first flagged.
Visit site →reactbits.dev

shadcn/ui

Component foundationFree · MIT

The de facto substrate every AI coding tool builds on.

  • The canonical copy-paste component system for React and Tailwind — plus a new 2026 chat suite (MessageScroller, Message, Bubble, Attachment).
  • Components are added via CLI as code you own in your repo, which is exactly why AI agents handle it so well — the model can read and edit every line.
  • Ships official Blocks (dashboards, auth, charts), a Directory of community registries, and shadcn/typeset for styled markdown.
  • As of 2026 any public GitHub repo can become an installable registry, and many AI tools ship shadcn registry support natively.
  • Changelog shows dated entries every month through July 2026, including the migration from Radix to Base UI and CLI v4.
  • Almost every other entry on this list (tweakcn, Magic UI, Aceternity blocks) plugs into it.
Why it wonNot an alternative but the baseline: the gravitational center of AI-assisted frontend work, with the best maintenance cadence in the category.
Visit site →ui.shadcn.com
